# 🍳 ZestyVerse - Recipe Sharing Platform

## 🚀 Overview

ZestyVerse is a modern, full-stack recipe sharing web application where
users can: - Discover delicious recipes - Create and submit their own
recipes - Save & like favorite recipes - Explore categorized food
content

Built with a **premium UI/UX**, inspired by Netflix-style layouts.

------------------------------------------------------------------------

## ✨ Features

### 👤 Authentication

-   User login/signup
-   Protected routes

### 🍽️ Recipes

-   Create new recipes
-   Admin approval system
-   View detailed recipes

### ❤️ Interaction

-   Like recipes
-   Save recipes
-   Personalized saved recipes page

### 🔍 Search

-   Live search with popup suggestions

### 🎨 UI/UX

-   Premium gradient theme
-   Smooth animations
-   Responsive design (mobile + desktop)

## ⚙️ Installation

### 1️⃣ Clone the repo

    git clone https://github.com/your-username/zestyverse.git
    cd zestyverse

### 2️⃣ Install dependencies

#### Client

    cd client
    npm install

#### Server

    cd server
    npm install

------------------------------------------------------------------------

## ▶️ Run Project

### Start backend

    cd server
    npm start

### Start frontend

    cd client
    npm start

------------------------------------------------------------------------

## 🌐 API Endpoints

### Recipes

-   GET `/api/recipes`
-   GET `/api/recipes/:id`
-   POST `/api/recipes`

### User Actions

-   POST `/api/recipes/:id/like`
-   POST `/api/recipes/:id/save`
-   GET `/api/recipes/saved/:userId`
